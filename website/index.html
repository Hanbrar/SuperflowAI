Hold Ctrl+Space and talk.

Simple voice dictation for free. Super Flow transcribes and pastes directly where your cursor is.

Download Windows App GitHub

Quick Start

  1. Open Super Flow.
  2. Select your microphone.
  3. Hold Ctrl+Space to start dictation.
  4. Release to transcribe and auto-paste text.

Features

Fast Dictation

Low-latency local speech-to-text for coding, emails, and rapid drafting.

Session PDF Export

Export temporary session transcripts to PDF whenever you need a record.

Privacy-First Flow

No required account and no forced permanent chat history for core usage.
