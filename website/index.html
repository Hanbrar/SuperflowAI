Windows Desktop App

Hold Ctrl+Space, talk, and innovate.

Simple voice dictation for free. Super Flow transcribes your speech and pastes it exactly where your cursor is.

Download Windows App View GitHub

How It Works

  1. Open Super Flow.
  2. Select your microphone.
  3. Hold Ctrl+Space to start dictation.
  4. Release to transcribe and auto-paste text.

Simple + Fast

Built for coding, email, and posting with low-latency local transcription and direct paste flow.

Session PDF

No permanent storage by default. Export your temporary session transcript to PDF anytime.

Always Free

Uses a local open-source speech stack, so there is no subscription for core dictation usage.
